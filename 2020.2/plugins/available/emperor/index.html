emperor¶

Description This QIIME 2 plugin wraps Emperor and supports interactive visualization of ordination plots.
Version 2020.2.0
Website http://emperor.microbio.me
Support Please post to the QIIME 2 forum for help with this plugin: https://forum.qiime2.org
q2cli Invocation 
qiime emperor
Artifact API Import 
from qiime2.plugins import emperor

  • Yoshiki Vázquez-Baeza, Meg Pirrung, Antonio Gonzalez, and Rob Knight. Emperor: a tool for visualizing high-throughput microbial community data. Gigascience, 2(1):16, 2013. doi:10.1186/2047-217X-2-16.
