regress_samples: Supervised learning regressor.
===============================================

Predicts a continuous sample metadata category using a supervised learning regressor. Splits input data into training and test sets. The training set is used to train and test the estimator using a stratified k-fold cross-validation scheme. This includes optional steps for automated feature extraction and hyperparameter optimization. The test set validates classification accuracy of the optimized estimator. Outputs classification results for test set. For more details on the learning  algorithm, see http://scikit-learn.org/stable/supervised_learning.html
