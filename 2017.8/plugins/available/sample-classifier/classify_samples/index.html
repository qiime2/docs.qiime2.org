classify_samples: Supervised learning classifier.Â¶

Predicts a categorical sample metadata category using a supervised learning classifier. Splits input data into training and test sets. The training set is used to train and test the estimator using a stratified k-fold cross-validation scheme. This includes optional steps for automated feature extraction and hyperparameter optimization. The test set validates classification accuracy of the optimized estimator. Outputs classification results for test set. For more details on the learning algorithm, see http://scikit-learn.org/stable/supervised_learning.html

Inputs
Name Type Default Description
table FeatureTable[Frequency] Required Feature table containing all features that should be used for target prediction.
Parameters
Name Type Default Description
metadata MetadataCategory Required Sample metadata category to use as prediction target.
test_size Float % Range(0.0, 1.0, inclusive_start=False) 0.2 Fraction of input samples to exclude from training set and use for classifier testing.
step Float % Range(0.0, 1.0, inclusive_start=False) 0.05 If optimize_feature_selection is True, step is the percentage of features to remove at each iteration.
cv Int % Range(1) 5 Number of k-fold cross-validations to perform.
random_state Int None Seed used by random number generator.
n_jobs Int 1 Number of jobs to run in parallel.
n_estimators Int % Range(1) 100 Number of trees to grow for estimation. More trees will improve predictive accuracy up to a threshold level, but will also increase time and memory requirements. This parameter only affects ensemble estimators, such as Random Forest, AdaBoost, ExtraTrees, and GradientBoosting.
estimator Str % Choices({'AdaBoostClassifier', 'ExtraTreesClassifier', 'GradientBoostingClassifier', 'KNeighborsClassifier', 'LinearSVC', 'RandomForestClassifier', 'SVC'}) RandomForestClassifier Estimator method to use for sample prediction.
optimize_feature_selection Bool False Automatically optimize input feature selection using recursive feature elimination.
parameter_tuning Bool False Automatically tune hyperparameters using random grid search.
