alpha_rarefaction: Alpha rarefaction curvesÂ¶

Generate interactive alpha rarefaction curves by computing rarefactions between min_depth and max_depth. The number of intermediate depths to compute is controlled by the steps parameter, with n iterations being computed at each rarefaction depth. If sample metadata is provided, samples may be grouped based on distinct values within a metadata column.

Inputs
Name Type Default Description
table FeatureTable[Frequency] Required Feature table to compute rarefaction curves from.
phylogeny Phylogeny[Rooted] None Optional phylogeny for phylogenetic metrics.
Parameters
Name Type Default Description
max_depth Int % Range(1) Required The maximum rarefaction depth. Must be greater than min_depth.
metric Str % Choices({'ace', 'berger_parker_d', 'brillouin_d', 'chao1', 'dominance', 'doubles', 'enspie', 'faith_pd', 'fisher_alpha', 'gini_index', 'goods_coverage', 'heip_e', 'lladser_pe', 'margalef', 'mcintosh_d', 'mcintosh_e', 'menhinick', 'michaelis_menten_fit', 'observed_otus', 'pielou_e', 'robbins', 'shannon', 'simpson', 'simpson_e', 'singles'}) None The metric to be measured. By default computes observed_otus, shannon, and if phylogeny is provided, faith_pd.
metadata Metadata None The sample metadata.
min_depth Int % Range(1) 1 The minimum rarefaction depth.
steps Int % Range(2) 10 The number of rarefaction depths to include between min_depth and max_depth.
iterations Int % Range(1) 10 The number of rarefied feature tables to compute at each step.
Outputs
Name Type Default Description
visualization Visualization N/A No description
