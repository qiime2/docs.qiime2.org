Computes a user-specified alpha diversity metric for all samples in a feature table.

Inputs
Name Type Default Description
table FeatureTable[Frequency] % Properties(['uniform-sampling']) Required The feature table containing the samples for which alpha diversity should be computed.
Parameters
Name Type Default Description
metric Str % Choices({'ace', 'berger_parker_d', 'brillouin_d', 'chao1', 'chao1_ci', 'dominance', 'doubles', 'enspie', 'esty_ci', 'fisher_alpha', 'gini_index', 'goods_coverage', 'heip_e', 'kempton_taylor_q', 'lladser_ci', 'lladser_pe', 'margalef', 'mcintosh_d', 'mcintosh_e', 'menhinick', 'michaelis_menten_fit', 'observed_otus', 'osd', 'pielou_e', 'robbins', 'shannon', 'simpson', 'simpson_e', 'singles', 'strong'}) Required The alpha diversity metric to be computed.
Outputs
Name Type Default Description
alpha_diversity SampleData[AlphaDiversity] N/A Vector containing per-sample alpha diversities.
