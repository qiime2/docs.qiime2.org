Available pluginsÂ¶

QIIME 2 microbiome analysis functionality is made available to users via plugins. The following official plugins are currently included in QIIME 2 train releases:

  • alignment
    • Methods
      • mafft: De novo multiple sequence alignment with MAFFT
      • mask: Positional conservation and gap filtering.
    • Visualizers
  • composition
    • Methods
      • add_pseudocount: Add pseudocount to table
    • Visualizers
      • ancom: Apply ANCOM to identify features that differ in abundance.
  • dada2
    • Methods
      • denoise: Denoise and dereplicate
    • Visualizers
      • plot_qualities: Plot positional qualitites
  • demux
    • Methods
      • emp: Demultiplex sequence data generated with the EMP protocol.
    • Visualizers
      • summarize: Summarize counts per sample.
  • diversity
    • Methods
      • alpha: Alpha diversity
      • alpha_phylogenetic: Alpha diversity (phylogenetic)
      • beta: Beta diversity
      • beta_phylogenetic: Beta diversity (phylogenetic)
      • core_metrics: Core diversity metrics.
      • pcoa: Principal Coordinate Analysis
    • Visualizers
      • alpha_correlation: Alpha diversity correlation
      • alpha_group_significance: Alpha diversity comparisons
      • beta_group_significance: Beta diversity group significance
      • bioenv: bioenv
  • emperor
    • Methods
    • Visualizers
      • plot: Visualize and Interact with Principal Coordinates Analysis Plots
  • feature-classifier
    • Methods
      • classify: Classify reads by taxon.
      • classify_paired_end: Classify reads by taxon.
      • extract_paired_end_reads: Extract reads from reference.
      • extract_reads: Extract reads from reference.
      • fit_classifier: Train a scikit-learn classifier.
      • fit_classifier_naive_bayes: Train the naive_bayes.MultinomialNB classifier.
      • fit_classifier_naive_bayes_paired_end: Train the naive_bayes.MultinomialNB classifier.
      • fit_classifier_paired_end: Train a scikit-learn classifier.
      • fit_classifier_svc: Train the svm.SVC classifier.
      • fit_classifier_svc_paired_end: Train the svm.SVC classifier.
    • Visualizers
  • feature-table
    • Methods
      • filter_features: Filter features from table.
      • filter_samples: Filter samples from table.
      • merge: Combine two tables
      • merge_seq_data: Combine two collections of feature sequences
      • merge_taxa_data: Combine two collections of feature taxonomies
      • presence_absence: Convert to presence/absence
      • rarefy: Rarefy table
      • relative_frequency: Convert to relative frequencies
    • Visualizers
      • summarize: Summarize table
      • tabulate_seqs: View sequence associated with each feature
  • phylogeny
    • Methods
      • fasttree: Construct a phylogenetic tree with FastTree.
      • filter_table: Remove features from table if they’re not present in tree.
      • midpoint_root: Midpoint root an unrooted phylogenetic tree.
    • Visualizers
  • taxa
    • Methods
      • collapse: Collapse features by their taxonomy at the specified level
    • Visualizers
      • barplot: Visualize taxonomy with an interactive bar plot
      • tabulate: View taxonomy associated with each feature
  • types
    • Methods
    • Visualizers
