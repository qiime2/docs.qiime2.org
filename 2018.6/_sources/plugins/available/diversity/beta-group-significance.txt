                                     Matrix of distances between pairs of

     --m-metadata-file MULTIPLE PATH

                                     Column from metadata file or artifact
                                     viewable as metadata. Categorical sample
                                     metadata column.  [required]
     --p-method [anosim|permanova]   The group significance test to be applied.
                                     [default: permanova]
     --p-pairwise / --p-no-pairwise  Perform pairwise tests between all pairs of
                                     groups in addition to the test across all
                                     groups. This can be very slow if there are a

                                     [default: False]
     --p-permutations INTEGER        The number of permutations to be run when
