taxa¶

Description This QIIME 2 plugin provides functionality for working with and visualizing taxonomic annotations of features.
Version 2017.6.0
Website https://github.com/qiime2/q2-taxa
Support Please post to the QIIME 2 forum for help with this plugin: https://forum.qiime2.org
Citation No citation available. Cite plugin website: https://github.com/qiime2/q2-taxa

Methods¶

  • collapse: Collapse features by their taxonomy at the specified level

Visualizers¶

  • barplot: Visualize taxonomy with an interactive bar plot
  • tabulate: View taxonomy associated with each feature
